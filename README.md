# Русификатор Sunderfolk
 
## 🎮 О проекте
Русификатор предназначен для замены оригинального польского текста на русский язык.  
Актуально для версии игры **Sunderfolk Patch 2.0.1**.

## 📂 Установка
1. Скачайте архив с файлами русификатора.
2. Распакуйте содержимое с заменой в папку:
   Sunderfolk\Sunderfolk_Data\StreamingAssets\aa\StandaloneWindows64\
3. Запустите игру выберите польский язык и наслаждайтесь русским интерфейсом.

## ⚠️ Важно
- Русификатор протестирован только с патчем **2.0.1**.  
- Возможны неточности перевода, так как используется машинный перевод.  
- Если вы заметили ошибки или хотите предложить улучшения — создавайте **Issues** или **Pull Requests**.
